Initialisation du project:

Angular CLI:

On va commencer par installer le CLI de Angular en globale sur notre machine avec la commande suivante (avec sudo si besoin):

  1. ~monPc@mcl/Desktop: npm install -g @angular/cli

Maintenant, la commande ng est disponible depuis la ligne de commandes depuis n'importe quel dossier de l'ordinateur

Pour plus d'informations, vous pourrez trouver la documentation de chaque élément sur le site corespondant:

  • node.js: https://node.js.org/en/docs/
  • npm: https://docs.npmjs.com/
  • ng: https://cli.angular.io/

Création du projet

Pour créer un nouveau projet Angular, naviguez vers le dossier souhaité depuis une ligne de commande et saisissez la commande suivante :

  1. ~monPc@mcl/Desktop: ng new clamer --skip-tests = true

Ensuite, navigez dans le dossier du projeet et lancez le serveur de developpement avec la commande suivante:

  1. ~monPc@mcl/Desktop: cd clamer

Ce qui va nous permettre de installer bootstrap dans notre nouveau projet, en exécutant la commande suivante:

  1. ~monPc@mcl/Desktop/clamer: npm install bootstrap --save

Une fois l'installation términer nous alons ouvrir notre project avec notre editer de text favoris. Visual studio code pour ma part:

  1. ~monPc@mcl/Desktop/clamer: code .

bootstrap est installer dans notre projet mais n'est pas encore intégrer à ce dernier.
